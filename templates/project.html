02Machine Learning

Machine Learning and Image Recognition

Key Concepts

    ML and Jupyter Notebooks Project
  • SciKit, Jupytr Notebooks, NumPy, Anaconda
  • KNeighbors Classifier, DecisionTreeClassifier (CSV)
  • reallysmartbrain project:
    Tensorflow, ImageAI (ImagePrediction)
  • SqueezeNet
Project Files:
    ML and Jupyter Notebooks:
    • reallysmartbrain:

About

There were two projects as part of the Machine Learning portion of the Zero to Mastery Course. The first project involved using Jupyter Notebooks along side the Iris Data Set from SciKit. The data was split into a training and test set and then the KNeighborsClassifer library was used to see how accurately the model could predict an input of an Iris. The same concept was then applied for the DecisionTree algorithm from scikit-learn. This project demonsrated the importance of a machine learning scientist's role in choosing the test and training data as well as the algorithm choice to achieve the best results.

The second project, called "reallysmartbrain", involved using the TensorFlow platform, ImagePrediction from ImageAI, and the SqueezeNet model (chosen mostly due to it's smaller size and other models were introduced). The model was then given the input of a few various images and then 5 results of what it identified and its probability are displayed.
